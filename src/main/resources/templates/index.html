This is instruction to use the link-generator

This service allows to shortening and expansion of links. The service is developed with the help of Heroku, Spring Boot, and H2 database.

Possible usage in mentioned lower:

http://link-generator-bitly.herokuapp.com/expand?url=https://bit.ly/3c0guSt/ will expand the link to it's long variant, and will return JSON with long URL via Bit.ly service.

http://link-generator-bitly.herokuapp.com/shorten?url=http://endorphina.com/ will shorten the link to it's short variant and will return JSON with short URL via Bit.ly service.
